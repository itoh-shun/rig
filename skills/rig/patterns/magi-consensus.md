# pattern: magi-consensus

MAGI の3号機（Melchior-1 / Balthasar-2 / Casper-3）が並列（`pattern: parallel-fanout`）に投じた票（可決 / 条件付可決 / 否決）を集約し、**多数決**で一意の判決を下す合議ゲート。`review-gate` の MAGI 版だが、集約規則が **majority-vote**（REJECT 1 件で即保留ではなく、多数で決める）である点が異なる。

## 概要

3つの号機 subagent から `output-contracts/magi-verdict` 形式で返る判定（`判定:` 行）を集計し、オーケストレーターが**決定論的に**最終判決を出す。各号機は独立した評価軸（Melchior=正しさ / Balthasar=守り / Casper=価値）に閉じて投票するため、3票は直交した観点を表す。判決は下表に従って機械的に決まる（2 回叩いても同じ＝determinism-by-gate）。

## 集計（判定行の機械抽出）

各 verdict の末尾（最後に現れる）`判定:` 行から票を抽出し、3 号機分を数える。

- **A** = 可決の数 ／ **C** = 条件付可決の数 ／ **R** = 否決の数（A + C + R = 3）
- いずれかの号機が明確な票を返さず**評価不能（データ不足で判定保留）**を表明した場合は、票を捏造せず**審議継続**へ（下表最終行）。

## 判決表（多数決）

| 集計 | 最終判決 | 対応 |
|---|---|---|
| R ≥ 2 | **否決** | 多数が否決。停止して否決理由（各否決号機の核心）を user に提示し、方針を委ねる。 |
| R = 1（残り 2 が可決/条件付） | **可決（多数決 2:1）** | 多数で可決。**否決した 1 号機の懸念を「保留事項」として必ず明示**する（握り潰さない）。C ≥ 1 なら統合条件も併記。 |
| R = 0 かつ C ≥ 1 | **条件付可決** | 否決なし。条件付の必須条件をすべて統合（→下記）して提示し、条件充足を着手の前提にする。 |
| R = 0 かつ C = 0（A = 3） | **可決（全会一致）** | 3 号機一致。条件なしで進行。 |
| いずれかが評価不能 | **審議継続（再審議要求）** | 判決を捏造しない。不足している情報・前提を user に明示して問い、揃ってから再合議する（MAGI の「審議中」状態）。 |

## 統合条件の扱い

`条件付可決` の各号機が出した `条件:` を収集し統合する。矛盾する条件（号機間で相互に矛盾する要求）は独断で解決せず user に提示する（`review-gate` の統合 conditions と同方針）。

## 正準出力（MAGI コンソール）

判決は**判定行を先頭**に置いた固定フォーマットで出す（機械抽出可能・テーマ表示はその後）。

```

判定: <可決（全会一致）|可決（多数決 2:1）|条件付可決|否決|審議継続>
集計: 可決 <A> / 条件付 <C> / 否決 <R>（3 号機中）

┌─ MAGI SYSTEM ─────────────────────────────┐
│  MELCHIOR · 1    [<可決|条件付|否決>]   科学者  │
│  BALTHASAR · 2   [<可決|条件付|否決>]   母      │
│  CASPER · 3      [<可決|条件付|否決>]   女      │
├───────────────────────────────────────────┤
│         判　決　:　<可決|否決|条件付|審議中>      │
└───────────────────────────────────────────┘

- MELCHIOR-1（科学者）<票> — <核心1行>
- BALTHASAR-2（母）<票> — <核心1行>
```

## オーケストレーターの実行手順

1. 3 号機の `magi-verdict` から `判定:` 行と `核心:` 行を抽出する（長文は親 context に引き込まない）。
2. A / C / R を数え、評価不能の有無を確認する。
3. 判決表を参照して最終判決を決定する（多数決）。
4. R = 1 の可決時は否決号機の核心を**保留事項**として必ず転記する。条件付があれば統合条件を仕分け（必須／矛盾は user へ）。
5. 正準出力（MAGI コンソール）を提示し、`次アクション` に従う。**否決・審議継続では先へ進めない**（停止して user へ）。
