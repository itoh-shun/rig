# instruction: merge

CI が通ったことを確認してスカッシュマージし、後片付けを行う。

## 手順

### ① CI ゲート

manifest の `ci` セクションで定義されたワークフロー・チェックが全て green になるまで待機する。定義がない場合はリポジトリのデフォルト required checks を参照する。CI が失敗した場合は implement または verify へ差し戻す。

### ② スカッシュマージ

CI ゲートを通過したら、スカッシュマージを実行する。マージコミットメッセージのフォーマットは manifest の `merge.commit-format` に従う。定義がない場合は PR タイトルをそのまま使う。

### ③ クリーンアップ

マージ完了後：

- フィーチャーブランチをリモートから削除する
- ローカルブランチを削除する（manifest の `merge.delete-local` が false の場合はスキップ）

### ④ 完了報告

変更内容・PR URL・マージコミット SHA を出力してフローを終了する。
