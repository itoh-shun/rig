# knowledge: spd-domain / spd-glossary（SPD用語集）

> **DESCRIPTIVE 層**。SPDドメインのユビキタス言語。persona・レビュー出力・生成物はこの用語に揃える。最終確認: 2026-07-24。

| 用語 | 意味 |
|---|---|
| **SPD** | Supply Processing and Distribution。院内物品の購買・在庫・供給・消費データ管理を一元化する仕組み／業務／サービス |
| **定数管理** | 部署ごと品目ごとに常備量（定数）を決め、消費分だけ補充する在庫方式。SPD運用の中核 |
| **定数見直し** | 消費実績に基づき定数を増減させる定期メンテナンス。過剰在庫と欠品の両方を抑える |
| **SPDカード（物品カード）** | 消費捕捉用のカード。使用時に回収しバーコードで読み取り、補充・請求データにする |
| **消化払い（預託在庫）** | 在庫の所有権を卸/事業者に残し、使用時点で病院の購入が成立する方式。consignment |
| **院内SPD / 院外SPD** | 自院職員による運用（自院運用型）／SPD事業者への外部委託（外部委託型） |
| **SPDセンター** | 中央倉庫。院内設置（院内型）と院外集中センターから複数病院へ供給する形（院外型）がある |
| **診療材料** | 注射器・カテーテル・縫合糸・ガーゼ等、診療で消費される医療材料 |
| **特定保険医療材料** | 保険診療で手技料と別に材料費を算定できる材料。償還価格は厚労省告示（材料価格基準）で規定 |
| **償還価格 / 償還差益** | 保険請求できる公定価格／償還価格と実購入価格の差。病院の材料収支を左右する |
| **請求漏れ** | 使用した償還材料が保険請求に反映されない損失。消費データとレセプトの突合で防ぐ |
| **物品マスタ** | 品目・規格・単位・価格・償還区分等を整備したマスタ。国内医療材料は約75万規格とされる |
| **GS1バーコード / GTIN** | 医療材料の国際標準識別コード。改正薬機法で表示義務化（2022年12月〜） |
| **トレーサビリティ** | ロット/シリアル単位で「誰に・いつ・何を使ったか」を追跡できること。回収・安全対応の基盤 |
| **セット化（キット化）** | 術式・処置別に材料を組み合わせて供給する物流加工 |
| **欠品率 / 定数遵守率 / 棚卸差異** | SPD運用品質の代表KPI |
| **臨時請求** | 定数外・定数不足時のスポット請求。多いほど定数設計か供給が破綻しているサイン |
| **日本SPD協議会** | 一般社団法人。SPD事業者の業界団体（1999年SPD研究会→2017年法人化）。https://www.spdjapan.org/ |
| **行政Watch** | 協議会活動の一つ。診療報酬改定・材料価格・医療政策の動向監視と会員への情報提供 |
