# verdict-format — 集約フォーマット（親用）

6視点の評価が揃ったら、判定行・根拠・懸念点だけを読んで（対象全文を親の文脈に抱え直さず）次の形式で集約する。前置き・挨拶は書かない。

```
## SPDステークホルダー・レビュー結果

総合判定: <GO|条件付きGO|要再検討>

| 視点 | 判定 | ひとことで |
|---|---|---|
| 病院経営層 | ◎○△× | （根拠1行） |
| 用度・材料部 | ◎○△× | （根拠1行） |
| 看護現場 | ◎○△× | （根拠1行） |
| SPD現場 | ◎○△× | （根拠1行） |
| SPD事業者経営 | ◎○△× | （根拠1行） |
| 卸・流通 | ◎○△× | （根拠1行） |

### 優先アクション（優先順）
1. （誰が・何を）
2. ...

### 情報不足（記述に足りず評価できなかった点）
- （あれば。追記を促す。なければ「なし」）
```

## 総合判定の決定ルール

上位優先で最初に当てはまる段を採る。

- **GO**: `×` が無く、`△` が1つ以下。
- **条件付きGO**: `×` が無く、`△` が2つ以上（優先アクションの実施を条件として明示する）。
- **要再検討**: いずれかの視点に `×`（重大欠落）がある、または**事実誤認**（制度・償還・薬機法まわり）が指摘されている。

> 情報不足が過半で評価不能な視点が多い場合は、総合判定の代わりに「記述不足のため評価保留」と明示し、不足項目の追記を促す。
