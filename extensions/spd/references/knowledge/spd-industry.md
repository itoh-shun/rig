# knowledge: spd-industry（業界構造・日本SPD協議会・ステークホルダー）

> **記述的（DESCRIPTIVE）知識**。SPD業界の構造、業界団体（日本SPD協議会）、登場するステークホルダー（ペルソナの原型）、行政動向、業界課題を記述する。
>
> 主な出典: 一般社団法人 日本SPD協議会（https://www.spdjapan.org/ ）。最終確認: 2026-07-24。

## 一般社団法人 日本SPD協議会

- **沿革**: 1999年3月17日、SPD業務に携わる事業者間の横断的な意見交換・研究の場として **SPD研究会** が発足。2017年12月1日に **一般社団法人 日本SPD協議会** として法人化。情報発信ページ等では「日本医療製品物流管理協議会（日本SPD協議会）」の名称も用いられる。
- **設立理念**: 「SPDの普及と発展を通じて、国民医療の向上に貢献する」。
- **ミッション**: 医療の質の向上と経営効率化を両面から支えるSPD事業の永続的な発展を通じて、健全な医療サプライチェーンの実現を支援する。SPD業務の質の向上を通して業界に貢献し、SPDサービス業務の地位を高める。
- **ビジョン**: SPD業務の価値を最大化し、医療業界から不可欠な存在と認められ、**各事業者が適正な利益を確保できる環境**の構築を支援する。
- **会員**: SPDサービス提供事業者が中心。法人化以降は医療機関および医療機関に勤務する個人も参画可能。理事長は菊地公明氏（前理事長: 笠原庸介氏）。
- **主な活動**: ① 人材育成（研修・資格的な教育）、② SPD業務全体の最適化・標準化研究、③ 病院関係団体との意見交換、④ 法制面の提案・行政への働きかけ、⑤ 情報発信（会報・調査資料）、⑥ **行政Watch**（診療報酬改定・材料価格制度・医療政策の動向監視）。

## 業界構造（サプライチェーン上の登場人物）

```
メーカー → 医療材料卸/ディーラー → SPD事業者（or 院内SPD部門） → 病院各部署（病棟・手術室・外来） → 患者
                                        ↑
                       行政（厚労省: 材料価格・薬機法・診療報酬）が制度面から全体を規定
```

| ステークホルダー | 立場・関心 |
|---|---|
| **病院経営層・事務長** | 経営効率（在庫・購入価格・委託費の総額）、経営指標、委託の費用対効果、監査対応 |
| **用度課・材料部（購買担当）** | 物品マスタ・価格交渉・契約管理・検収。SPD事業者の現場カウンターパート |
| **看護部（師長・現場看護師）** | 欠品ゼロ・臨時請求の少なさ・補充の確実さ。物品業務の負担軽減が最大の関心 |
| **手術室・診療部門** | 術式別セット・預託材料（高額償還材料）の確実な供給とロット管理 |
| **SPD事業者（現場: センター長・オペレーター）** | 定数遵守率・欠品率・棚卸精度などの運用KPI、人員確保、現場との信頼関係 |
| **SPD事業者（経営）** | 契約の採算性（適正価格での受注）、人材育成、差別化（データ活用・DX提案） |
| **医療材料卸・ディーラー** | 納品精度・預託在庫の負担・価格交渉。SPD事業者と協調も競合もする |
| **メーカー** | 製品供給・バーコード表示（GS1）・回収時のトレーサビリティ |
| **行政（厚生労働省）** | 材料価格基準（償還価格）、診療報酬改定、薬機法（バーコード義務化）、医療DX、統一データベース |

## 行政動向（行政Watchの主対象）

- **診療報酬改定（2年ごと）**: 特定保険医療材料の償還価格見直し。償還価格の引き下げは病院の材料収支とSPD価格交渉に直結する。
- **改正薬機法のバーコード表示義務化**（2022年12月〜）と、厚労省の**医療機器等統一データベース**（2027年度運用開始予定）: 受発注・在庫管理の標準化と省人化を後押しする。
- **医師・医療従事者の働き方改革／タスクシフト**: 物品業務の外部化ニーズを高める追い風。
- **物流2024年問題・人手不足**: 配送・院内搬送の人員確保コスト上昇。SPD事業者の採算と、委託費の適正化議論に影響する。
- **医療DX（電子処方箋・標準型電子カルテ等）**: 消費データと診療データの接続、症例別原価計算への発展余地。

## 業界課題（協議会が問題提起する論点）

1. **委託費の適正化**: SPDは「安く請け負って当然」と見られがちで、過当競争により事業者の採算が悪化しやすい。ビジョンの「適正な利益を確保できる環境」はこの裏返し。
2. **契約範囲の曖昧さ**: 業務範囲（物流のみか、購買代行・価格交渉・データ提供まで含むか）が契約で明確にされず、追加業務が無償で膨らむ構造問題。
3. **人材確保・育成**: 医療材料の専門知識（75万種の規格・償還区分）を要する現場人材の不足。協議会の人材育成活動の背景。
4. **データ標準化の遅れ**: 病院ごとに異なるマスタ・システムがデータ活用と事業者間の相互運用を阻む。GS1・統一DBが解決の方向。
5. **在庫リスクの偏在**: 預託（消化払い）方式で在庫リスクが卸・事業者側に偏る商慣行。
